## 🎯 prerequisites

<b>Install Node  </b></br> 
node -v </br>

<b>Install NPM </b></br>
npm -v  </br>

<b>Clone project </b></br>

<b>Install dependencies package.json </b> </br>
 $ npm install</br>
 
 Done ✔️



## 💡 Use

<b>Run test cypress gui </b></br>
$ npx cypress open</br>

<b>Run test cypress headless</b></br>
$ npm run cypress:run</br>
